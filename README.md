# Prog-project
A Python-based retail management system with both a Command-Line Interface (CLI) and a Streamlit Web GUI. RetailTrack helps small store owners manage their product catalogue, process sales, track transactions, and generate business reports — all with automatic CSV persistence.

✨ Features

📦 Product Management — Add, update, restock, and remove products
💰 Sales Processing — Record sales with automatic stock deduction and receipt generation
📋 Transaction History — View all transactions with filtering by SKU or date range
📊 Reports & Analytics — Inventory reports, low-stock alerts, profit margins, and performance summaries
💾 Auto-Save — All data is persisted to CSV files automatically after every operation
🌐 Dual Interface — Use the terminal CLI or the browser-based Streamlit GUI
